Timeago.js

Simple library (less than 2 kb) that is used to format datetimes.

Set relative date JavaScript
var timeagoInstance = timeago('2016-06-10 12:12:12'); // set the relative date here.
timeagoInstance.format('2016-06-12', 'zh_CN');
Use timestamp JavaScript
timeago().format(Date.now() - 11 * 1000 * 60 * 60); // returns '11 hours ago'
Automatic rendering HTML
<time class="timeago" datetime="2016-06-30 09:20:00"></time>
